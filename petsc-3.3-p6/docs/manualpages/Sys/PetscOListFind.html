PetscOListFind

 givn a name, find the matching object

Synopsis

PetscErrorCode  PetscOListFind(PetscOList fl,const char name[],PetscObject *obj)

Input Parameters

fl - pointer to list
name - name string

Output Parameters

ob -the PETSc object

Notes

The name must have been registered with the PetscOListAdd() before calling this routine.

The reference count of the object is not increased

See Also

PetscOListDestroy(), PetscOListAdd(), PetscOListDuplicate(), PetscOListReverseFind(), PetscOListDuplicate()
