PetscGlobalSum

 Computes the sum over several processors. Only for use with ADIC!

Synopsis

PetscErrorCode  PetscGlobalSum(MPI_Comm comm, const PetscScalar* local,PetscScalar* result)
Collective on MPI_Comm

Input Parameters

local - the local value
comm - the processors that find the sum

Output Parameter

result -the sum

Notes

These functions are to be used ONLY inside user functions that are to be processed with ADIC. PETSc will automatically provide differentiated versions of these functions

See Also
