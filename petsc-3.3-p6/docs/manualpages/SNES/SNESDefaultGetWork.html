SNESDefaultGetWork

 Gets a number of work vectors.

Synopsis

#include "petscsnes.h"   
PetscErrorCode SNESDefaultGetWork(SNES snes,PetscInt nw)

Input Parameters

snes -the SNES context
nw -number of work vectors to allocate

Notes

Call this only if no work vectors have been allocated
