MatDAADSetSNES

 Tells the matrix what SNES it is using for the base U.

Synopsis

#include "petscdmda.h"    
#include "petscsnes.h"  
PetscErrorCode  MatDAADSetSNES(Mat A,SNES snes)
Logically Collective on Mat and SNES

Input Parameters

mat - the matrix
snes - the SNES

Notes: this is currently turned off for Fortran usage
